Regional Settings

Choose the country first. Country selection sets the default timezone and interface language, just like a phone country code selector. You can then adjust the interface language separately if needed.

Team Operation explanation

Standard mode is for one responsible person. Team mode is for several responsible people who can record actions for the same patient. In History, every record must show the actual action time, the role, and the email of the actor.

Time is always shown in 24-hour format. Each planned slot has its own live status: waiting before the slot, overdue after the slot until the next slot, and actual confirmation status when an action is recorded.
